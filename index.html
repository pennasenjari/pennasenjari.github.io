Työajanseurantasovellus (Worktime Tracking Application)

Click the screenshots for larger version

This is a Web application I programmed for a transport company few years ago. I did all the programming alone based on the customer's layout pictures. The site has two user types, admin (company manager) and user (truck driver).

The driver logs into their view and gives starting location when starting a shift, and later clicks the appropriate buttons to switch between task types (Waiting / Driving / Terminal Work / Break). After the drive the final location is also saved. The driver's views are programmed with plain HMTL, CSS and JavaScript.

The admin UI is programmed with ReactJS. In admin it's possible to view, modify and accept individual driver's accumulated hours, as well as handle drivers as groups and display total hours. The system automatically calculates driver's daily allowances (päivärahakertymä). It's possible to export the data formatted to Excel.

The backend runs on Laravel framework and relies on MySQL database to store the data. In addition, there is a link from Laravel to WordPress-based company intranet, which holds the user data used for authentication.

The program code belongs to my customer so unfortunately I'm now allowed to display the source code here.
